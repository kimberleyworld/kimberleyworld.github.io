Snake Sweeper

11/21

JavaScript Game (SCRUM project - team of 8)

JavaScript, HTML, CSS

A game where users must find all the lynx in the forest before they run out of lives or get bitten by too many snakes. The size of the forest and the amount of lynx is defined by the user and is randomly generated at the start of each game. We also added functionality to keep track of the current score and remaining lives.

Build

We created this app in OOP applying SOLID principles where possible. I worked on designing, creating and implementing the wireframes.
