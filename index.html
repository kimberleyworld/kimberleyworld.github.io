We created this app in OOP applying SOLID principles where possible. I worked on designing, creating and implementing the wireframes. As well as the hydrator/entity classes to receive and display data from a SQL database. The project utilised autoloading and namespacing throughout, validated and sanitised all user inputs before database insertion and unit tested our methods with PHPUnit.
